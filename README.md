# README

* Ruby version:
  2.3.3
 
* Rails version: 
  5.0.2  

* System dependencies: \n
  * PostgreSQL \n
  * Bootstrap \n
  * Carrierwave \n
  * Sendgrid - Account/API Keys \n
  * Cloudinary - Account/API Keys \n

* Database creation
  From your cloned directory:
  * run `bundle` 
  * run `rails db:create`
 
 * Database initialization
    *Run: `rails db:migrate`
    *Finally, run: `rails db:seed`
