# README

* Ruby version:
  2.3.3
 
* Rails version: 
  5.0.2  

* System dependencies:
  *PostgreSQL
  *Bootstrap
  *Carrierwave
  *Sendgrid - Account/API Keys
  *Cloudinary - Account/API Keys

* Database creation
  From your cloned directory:
  *run `bundle` 
  *run `rails db:create`
 
 * Database initialization
    *Run: `rails db:migrate`
    *Finally, run: `rails db:seed`
